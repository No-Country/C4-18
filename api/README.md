## Correr el proyecto

Instalar Node.js. 

Clonar el proyecto.
- [C4-18](https://github.com/No-Country/C4-18)

Una vez clonado el proyecto ejecutar los siguientes comandos:

npm install

npm start o npm run dev si estas en entorno de desarrollo.
